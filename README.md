# 月球基地建设综合决策优化模型 (LBC-CDO Model)

## 项目概述

本项目是一个针对月球基地建设的综合决策优化模型，涵盖了从风险管理、运输系统优化、水资源管理到生命周期评估等多个方面。模型采用多目标多阶段优化方法，通过动态规划、NSGA-II遗传算法等技术，为月球基地建设提供科学的决策支持。

## 各文件夹用途

### 2026/
- **结果/**: 存放2026年分析的所有结果文件，包括图表和数据。
- **问题/**: 存放2026年的问题描述文件（PDF格式）。
- **脚本/**: 存放2026年的分析脚本，包括各种优化和可视化脚本。
- **笔记本/**: 存放2026年的Jupyter笔记本文件。

### 数学建模/
- **数据/**: 存放建模过程中使用的数据文件，包括CSV格式的参数和结果。
- **问题/**: 存放问题描述、分析报告和方法论文档（Markdown格式）。
- **脚本/**: 存放建模和分析的核心脚本，包括各种算法实现。
- **可视化/**: 存放建模过程中生成的可视化结果，包括各种图表。
- **第一题动态规划/**: 存放第一题动态规划相关的文件和结果。

### 根目录其他文件夹
- **数据/**: 存放通用数据文件，如原始数据和模型表面数据。
- **问题/**: 存放通用问题描述文件和分析报告。
- **火箭预测/**: 存放火箭发射预测相关的脚本和结果。
- **脚本/**: 存放通用脚本文件，如数据处理和分析工具。
- **可视化/**: 存放通用可视化结果。

## 如何运行代码

### 1. 环境准备
- Python 3.13
- 主要依赖库：numpy, pandas, matplotlib, seaborn, scipy

### 2. 运行步骤
1. 进入对应文件夹
2. 运行相关脚本文件
   - 例如：`python 脚本/创建3D帕累托图.py`

### 3. 主要脚本说明
- **第一题双目标TOPSIS.py**: 第一题双目标优化和TOPSIS分析
- **第二题敏感性分析LHS.py**: 第二题敏感性分析
- **第三题水资源策略.py**: 第三题水资源策略分析
- **第四题生命周期评估NSGA2.py**: 第四题生命周期评估和NSGA-II优化

## 注意事项

1. **文件路径**：所有脚本中的文件路径均为相对路径，请确保在正确的目录下运行。
2. **数据文件**：部分脚本依赖于特定的数据文件，请确保数据文件存在。
3. **计算资源**：部分优化算法可能需要较长时间运行，请耐心等待。
4. **结果保存**：大多数脚本会自动将结果保存到指定目录，无需手动操作。

## 模型特点

- **多目标优化**：同时考虑成本、时间、环境等多个目标
- **多阶段决策**：分为营地建设期、基地扩展期和城市发展期
- **混合算法**：结合动态规划、NSGA-II遗传算法等多种方法
- **系统工程**：涵盖运输、水资源、能源等多个子系统
- **决策支持**：为政策制定者提供优化方案
