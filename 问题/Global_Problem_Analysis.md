# 1.2 问题分析 (Problem Analysis)

为了确保 10 万人月球殖民地建设工程的可行性与长期可持续性，我们的总体目标是建立一个集成的系统工程模型，以在多阶段建设中平衡成本、时间、风险与资源约束。为了实现这一目标，我们需要按照以下逻辑递进解决问题：

1.  **开发一个多阶段物流规划与动态优化模型**
    (对应问题 1：成本与时间)
    我们需要针对 100 万吨物资的巨量运输任务，建立一个多目标优化模型。该模型需量化不同运输载体（太空电梯 Galactic Harbour vs 传统火箭）在“营地-基地-城市”三个发展阶段的投入产出比，以**最小化总成本**和**最短化建设周期**为目标，制定最优的物资分配与运输调度计划。

2.  **进行敏感性分析与风险鲁棒性评估**
    (对应问题 2：风险分析)
    我们需要评估模型在不确定环境下的稳定性。通过引入随机干扰变量（如太空电梯缆索摆动、火箭发射失败率、天气延误），对基础模型进行敏感性分析。这将帮助我们识别影响项目成败的关键风险因子，并确定模型的**鲁棒性边界**，使决策者能够预判极端情况下的备用方案。

3.  **调整模型以适应资源制约，并改进环境得失平衡**
    (对应问题 3 & 4：水资源与环境)
    我们需要将模型从单纯的“建设物流”扩展到长期的“生存维系”与“全球影响”层面：
    *   **适应性评估 (水资源)**：将模型应用于殖民地运行期的水资源代谢分析。结合我们预测的**2050年全球火箭发射上限（约 7300 次/年）**，评估单纯依赖地球补给的可行性，从而论证向原位资源利用（ISRU）模式转型的数学必然性。
    *   **策略改进 (环境)**：评估大规模太空活动对地球环境的潜在冲击。通过权衡环境成本与开发速度，对模型进行改进，以寻求人类地外扩张与地球母星生态保护之间的最佳平衡点。
